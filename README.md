## Integrated Output
To use the integrated output on vscode, you will need to put this script to the `autoexec` folder:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/Deepsn/synapsex-tools/main/OutputRedirect.lua"))()
```
